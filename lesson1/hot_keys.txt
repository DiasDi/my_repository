Которые показывал на уроке:

Win + R 	- открыть окно "Выполнить"
	1) cmd 		- команда для окна "Выполнить" для открытия консоли windows

	'python' 	- команда для запуска python оболочки в консоли windows
	"clear"		- очистка командной строки
Новые команды:

Просто находясь на рабочем столе или внутри папки через проводник нажмите следующие клавиши:
Ctrl+Shift+N 	- создать папку (не забыть дать сразу название для нее)

Не забываем про правила наименования папок:
1) на латинице
2) без пробелов (вместо пробела использовать знак подчеркивания _ )

Ctrl + / 				- закомментировать часть кода
Ctrl + Shift + ` 		- запустить терминал внутри VS Code
F2 		 				- переименовать файл
Ctrl + S	 			- сохранить файл
